#OOP Project
C++ Hospital Management System project using SFML 
##Github Repository
https://github.com/hibaatif05/OOP-Project
Requirements
Windows 10 or later
Visual Studio 2022
SFML Library

Installing SFML
Download SFML (MSVC version) from sfml-dev.org
Extract it so the folders are placed as:
C:\SFML\include
C:\SFML\lib
Copy the SFML .dll files into the same folder as the .exe after building

How to Compile
Open OOP PROJECT.sln in Visual Studio 2022
Select Debug or Release from the configuration dropdown
Go to Build, Build Solution (or press Ctrl + Shift + B)

How to Run
Press F5 in Visual Studio
